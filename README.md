# Paper Test #Angular

## Getting Started

* Dependencies 
 ```
"@angular/cli": "~11.1.2"
node > 11 prefered v14.15.4
latest chrome version 
```

* Install the packages and you're ready to go.
```
 npm install
```
*  Run the application:
```
Run `npm start or ng serve` for a dev server. 
Navigate to `http://localhost:4200/`.
```
